# CVs

Coloca aquí tus archivos PDF de CV.

Ejemplos de nombres:
- `CV-Samuel-Ospina-ES.pdf`
- `CV-Samuel-Ospina-EN.pdf`

Luego actualiza la lista en `js/cv.js` (array `cvs`) para que aparezcan en la página `cv.html`.

> Nota: En un sitio estático (GitHub Pages, Netlify, etc.) JavaScript no puede “listar” automáticamente los archivos de una carpeta. Por eso se define el listado manualmente en `js/cv.js`.
